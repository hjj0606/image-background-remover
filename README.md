# 🚀 AI 一键智能抠图工具 (AI Background Remover)

这是一个基于 Python Flask 和 AI 模型开发的开源背景去除工具。用户只需上传图片，即可实现毫秒级的背景擦除，并下载透明背景的 PNG 图片。

## ✨ 功能特点
- **完全免费**：无需支付 API 费用。
- **本地处理**：基于 `rembg` 开源模型，保护隐私。
- **一键下载**：处理完成后直接下载高清透明图。
- **响应式 UI**：适配手机与电脑端。

## 🛠️ 技术栈
- **后端**: Python (Flask)
- **AI 核心**: rembg (基于 U-Net 架构)
- **前端**: HTML5, JavaScript, Tailwind CSS
